# camera
projet camera

## Configuration `.env`
1. Copier le fichier d'exemple:
   ```bash
   cp .env-example .env
   ```
2. Modifier les valeurs selon votre réseau (ex: `ESP32_IP`, `MJPEG_URL`) et vos identifiants.
3. Lancer le projet avec Docker Compose:
   ```bash
   docker compose up --build
   ```
